# Jenga365 AI Platform
**Strategic Theme (2026-2027): Consistency, Growth, and Impact**

Jenga365 is a pioneering dual-engine development initiative positioned at the intersection of high-performance sport and sustainable community development. We bridge the gap between potential and performance by keeping athletes constructive 365 days a year.

## 🏁 The Dual-Engine Model
Our operational framework runs on two synchronized engines:
- **Engine A: Human & Social Capital Development** — Mental resilience, financial literacy, and professional networks for the "Total Athlete."
- **Engine B: Environmental & Social Stewardship** — Mobilizing disciplined athletes for environmental restoration ("The Green Game") and community welfare.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL (Neon)
- Sanity.io Account

### Development
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the platform.
